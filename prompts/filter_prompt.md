# 二次筛选指令

## 任务
分析用户原始问题和百科搜索返回的词条列表，选择最相关的一个词条。

## 分析流程
1.  理解用户的原始问题意图。
2.  逐一评估词条列表中的每个标题，判断其与用户问题的相关性。
3.  选择一个最能直接回答用户问题或提供最核心背景知识的词条。

## 输出要求
严格按照以下 JSON 格式输出，不要包含任何其他文字。

```json
{
  "selected_title": "选择的词条标题"
}
```

如果认为所有词条都不相关，则输出：
```json
{
  "selected_title": null
}
```
